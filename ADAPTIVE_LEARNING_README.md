# 自适应学习模块使用说明

## 概述

自适应学习模块通过实时监控行情走势和交易表现，自动分析并给出优化建议，帮助提高交易系统的收益率。

## 核心功能

### 1. 市场行情分析
- **波动率分析**：计算市场ATR，识别高波动/低波动市场
- **趋势强度分析**：通过MA斜率判断市场趋势方向
- **成交量趋势分析**：识别成交量变化趋势
- **市场阶段分类**：trending_up / trending_down / sideways / volatile

### 2. 交易表现分析
- **基础指标**：胜率、盈亏比、平均盈利/亏损、持仓时间
- **风险指标**：夏普比率、最大回撤
- **分类表现**：各分类的胜率、平均盈亏
- **近期表现**：最近N笔交易的胜率（用于识别趋势变化）

### 3. 优化建议生成

#### 止损优化
- **收紧止损**：当胜率低且平均亏损大时，建议收紧止损
- **放宽止损**：高波动市场且胜率较高时，建议放宽止损避免过早止损

#### 止盈优化
- **降低止盈**：平均盈利远小于止盈目标时，建议降低止盈提高止盈率
- **提高止盈**：趋势强劲且胜率高时，建议提高止盈获取更大利润

#### 信号阈值优化
- **提高阈值**：胜率低时，建议提高信号阈值过滤低质量信号
- **降低阈值**：胜率高但交易频率低时，建议降低阈值增加交易机会

#### 分类权重优化
- **增加权重**：表现优秀的分类（胜率>60%，平均盈亏>0）
- **降低权重**：表现较差的分类（胜率<40%，平均盈亏<0）

#### 持仓时间优化
- **延长持仓**：平均持仓时间过短且胜率低时，建议延长持仓时间

## 配置

在 `config.env` 中配置：

```env
# 学习分析间隔（分钟），默认60分钟（每小时分析一次）
LEARNING_ANALYSIS_INTERVAL_MINUTES=60
```

## 使用方法

### 自动运行
模块已集成到 `LiveTraderV3`，会在每个监控周期自动运行（根据配置的间隔）。

### 手动触发
可以通过修改 `LEARNING_ANALYSIS_INTERVAL_MINUTES=0` 来让每次监控周期都运行分析。

## 分析结果保存

分析结果会自动保存到 `learning_analysis.json` 文件中，包含：
- 时间戳
- 所有优化建议的详细信息

## 注意事项

1. **最少交易次数**：需要至少10笔交易才会生成建议（可在代码中调整 `min_trades_for_analysis`）
2. **置信度**：建议的置信度基于数据质量和分析结果，建议优先考虑高置信度的建议
3. **优先级**：优先级1-5，1为最高优先级，建议优先实施
4. **手动确认**：所有建议都需要手动确认后实施，系统不会自动修改参数

## 未来扩展

可以进一步扩展的功能：
1. **机器学习模型**：使用历史数据训练模型预测最佳参数
2. **A/B测试**：自动测试不同参数组合的效果
3. **实时参数调整**：在确认安全的情况下，自动应用部分优化建议
4. **多策略对比**：对比不同策略参数的表现
