## 使用方式

### 虚拟交易模式（默认）

```env
# config.env
TRADE_MODE=virtual
TRADE_ENABLED=true
```

```bash
python live_trader_v3.py
```

### 实盘交易模式

**重要：启用前请仔细阅读 `REAL_TRADING_GUIDE.md`！**

```env
# config.env
TRADE_MODE=real
TRADE_ENABLED=true
BINANCE_API_KEY=your_actual_api_key
BINANCE_SECRET_KEY=your_actual_secret_key
```

```bash
python live_trader_v3.py
```

系统会：
1. 验证API密钥配置
2. 显示警告信息
3. 5秒倒计时确认
4. 启动实盘交易

## 注意事项

1. **实盘交易前必须：**
   - 在虚拟模式下充分测试（至少1-2周）
   - 使用小资金进行实盘测试
   - 配置钉钉通知
   - 理解所有风险

2. **API密钥安全：**
   - 只授予"启用交易"权限
